Fizemos o open source do React há mais de uma década para ajudar os desenvolvedores a criar ótimas experiências de usuário. Desde seus primeiros dias, o React recebeu contribuições substanciais de colaboradores fora da Meta. Com o tempo, o número de contribuidores e o escopo de suas contribuições cresceram significativamente. O que começou como uma ferramenta desenvolvida para a Meta se expandiu para um projeto que abrange várias empresas com contribuições regulares de todo o ecossistema. O React cresceu além dos limites de qualquer empresa.

Para servir melhor à comunidade React, estamos anunciando nossos planos de mover o React e o React Native da Meta para uma nova React Foundation. Como parte dessa mudança, também implementaremos uma nova estrutura de governança técnica independente. Acreditamos que essas mudanças nos permitirão fornecer mais recursos aos projetos do ecossistema React.

## A React Foundation {/*the-react-foundation*/}

Faremos da React Foundation a nova casa para React, React Native e alguns projetos de suporte como JSX. A missão da React Foundation será apoiar a comunidade e o ecossistema React. Uma vez implementada, a React Foundation irá:

* Manter a infraestrutura do React como GitHub, CI e marcas registradas
* Organizar a React Conf
* Criar iniciativas para apoiar o ecossistema React, como suporte financeiro a projetos do ecossistema, emissão de bolsas e criação de programas

A React Foundation será governada por um conselho de diretores, com Seth Webster atuando como diretor executivo. Este conselho direcionará fundos e recursos para apoiar o desenvolvimento, a comunidade e o ecossistema do React. Acreditamos que esta é a melhor estrutura para garantir que a React Foundation seja neutra em relação a fornecedores e reflita os melhores interesses da comunidade.

Os membros corporativos fundadores da React Foundation serão Amazon, Callstack, Expo, Meta, Microsoft, Software Mansion e Vercel. Essas empresas tiveram um grande impacto nos ecossistemas React e React Native e somos gratos por seu apoio. Estamos animados para receber ainda mais membros no futuro.

## Governança técnica do React {/*reacts-technical-governance*/}

Acreditamos que a direção técnica do React deve ser definida pelas pessoas que contribuem e mantêm o React. À medida que o React se move para uma fundação, é importante que nenhuma empresa ou organização seja super-representada. Para conseguir isso, planejamos definir uma nova estrutura de governança técnica para o React que seja independente da React Foundation.

Como parte da criação da nova estrutura de governança técnica do React, entraremos em contato com a comunidade para obter feedback. Uma vez finalizado, compartilharemos os detalhes em uma postagem futura.

## Obrigado {/*thank-you*/}

O incrível crescimento do React é graças a milhares de pessoas, empresas e projetos que moldaram o React. A criação da React Foundation é uma prova da força e vitalidade da comunidade React. Juntas, a React Foundation e a nova governança técnica do React garantirão que o futuro do React seja seguro nos próximos anos.
