title: Diretivas
---

<Intro>
As diretivas do React Compiler são literais de string especiais que controlam se funções específicas são compiladas.
</Intro>

```js
function MyComponent() {
  "use memo"; // Habilita esta compilação de componente
  return <div>{/* ... */}</div>;
}
```

<InlineToc />

---

## Visão geral {/*overview*/}

As diretivas do React Compiler fornecem controle preciso sobre quais funções são otimizadas pelo compilador. Elas são literais de string colocados no início do corpo de uma função ou no topo de um módulo.

### Diretivas disponíveis {/*available-directives*/}

*   **[`"use memo"`](/reference/react-compiler/directives/use-memo)** - Habilita a compilação de uma função
*   **[`"use no memo"`](/reference/react-compiler/directives/use-no-memo)** - Desabilita a compilação de uma função
